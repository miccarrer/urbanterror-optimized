# Credits

Urban Terror Optimized stands on the shoulders of decades of open-source work.

## Lineage

```
id Software — Quake III Arena (GPL v2)
 └── ioquake3 (ioq3) — community maintenance, portability
      └── ec-/Quake3e — renderer/VM/netcode optimizations, Vulkan renderer
           ├── omg-urt/urbanterror-slim — Urban Terror feature set
           └── urbanterror-optimized — this project
```

## Upstream projects

- **id Software** — original Quake III Arena engine, released under the GPL v2.
  See [docs/legal/id-readme.txt](legal/id-readme.txt).
- **[ioquake3](https://github.com/ioquake/ioq3)** — the community fork this lineage builds on.
- **[ec-/Quake3e](https://github.com/ec-/Quake3e)** by Eugene C. — the direct upstream;
  the bulk of the engine optimizations and the Vulkan renderer.
- **[Quake-III-Arena-Kenny-Edition](https://github.com/kennyalive/Quake-III-Arena-Kenny-Edition)**
  — original basis of the Vulkan renderer.
- **[omg-urt/urbanterror-slim](https://github.com/omg-urt/urbanterror-slim)** — reference for
  the Urban Terror specific features.
- **[OpenArena](https://github.com/OpenArena/engine)** and
  **[cnq3](https://bitbucket.org/CPMADevs/cnq3)** — additional renderer ideas.
